# Security Policy

## Reporting a vulnerability

Please do **not** open a public issue for security reports.

Instead, report privately via GitHub Security Advisories:
- https://github.com/unisone/intel-skill/security/advisories/new

Include:
- a clear description of the issue and impact
- steps to reproduce / proof of concept
- affected versions or commit hash

## Scope

This repository primarily contains prompt/code templates for a Claude Code skill.

If you find anything that could cause:
- secret leakage (tokens/keys/cookies)
- unsafe file operations
- unexpected network access
